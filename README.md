# CVM Tabel

Ferramentas para coleta e análise de emissões públicas registradas no portal SRE da CVM.

## Projetos

### [`cvm_emissoes/`](cvm_emissoes/README.md)

Interface Streamlit que coleta dados de emissões diretamente da API REST do portal SRE da CVM,
estrutura os dados por série e exporta para Excel. Inclui assistente de análise via Gemini.

**Acesse a documentação completa em [cvm_emissoes/README.md](cvm_emissoes/README.md).**
